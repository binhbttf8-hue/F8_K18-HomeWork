# 🛒 ShopAdmin – Hệ Thống Quản Lý Bán Hàng (JavaScript Thuần - K18)

## 📌 Giới thiệu

**ShopAdmin** là dự án xây dựng hệ thống quản lý bán hàng sử dụng JavaScript thuần (Vanilla JS), không sử dụng framework.

Dự án được thực hiện bởi sinh viên K18 nhằm mục tiêu:
- Thực hành thao tác DOM
- Xây dựng CRUD bằng JavaScript thuần
- Quản lý dữ liệu với LocalStorage
- Xây dựng giao diện Admin Dashboard thực tế

---

## 🚀 Tính năng chính

### 📊 1. Dashboard (Tổng quan)
- Hiển thị tổng doanh thu
- Tổng số đơn hàng
- Lợi nhuận
- Khách hàng mới
- Biểu đồ doanh thu 7 ngày gần nhất
- Biểu đồ cơ cấu sản phẩm

### 📦 2. Quản lý sản phẩm
- Thêm sản phẩm
- Sửa sản phẩm
- Xóa sản phẩm
- Tìm kiếm sản phẩm
- Lưu trữ dữ liệu bằng LocalStorage

### 🧾 3. Quản lý đơn hàng
- Tạo đơn hàng mới
- Cập nhật trạng thái đơn hàng
- Xem chi tiết đơn
- Tính tổng tiền tự động

### 👥 4. Quản lý khách hàng
- Thêm khách hàng
- Cập nhật thông tin
- Tìm kiếm khách hàng
- Thống kê khách hàng mới

### 📈 5. Báo cáo doanh thu
- Lọc theo ngày
- Thống kê doanh thu
- Thống kê lợi nhuận
- Biểu đồ trực quan

## 🛠 Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript (ES6+)

- Chart.js
